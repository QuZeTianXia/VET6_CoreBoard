    从学校带回家的件都焊上了，vet6芯片倒是能下载程序，但是引脚电平控制得不正常，分析原因可能有三个：    
    > 1、焊锡没上好，接触不良。  
      2、没带22pf的起振电容，时钟不正常。   
      3、boot0的接法有问题，导致不能从Flash启动系统。 

    前两个问题还容易解决，要是第三个问题恐怕板子要飞线了。
